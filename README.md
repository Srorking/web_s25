# دليل فعاليات الجامعة الافتراضية السورية

## 📌 وصف المشروع
تم تطوير هذا الموقع كجزء من متطلبات مادة تقنيات الويب الأساسية (BWP401)، وهو عبارة عن دليل لعرض الفعاليات التي تقام ضمن الجامعة الافتراضية السورية (Syrian Virtual University)، بهدف تسهيل وصول الطلاب إلى المعلومات المتعلقة بالأنشطة المختلفة.

---

## 🎯 هدف الموقع
يهدف الموقع إلى:
- عرض الفعاليات بطريقة منظمة وسهلة الاستخدام
- تمكين المستخدم من البحث والتصفية حسب التصنيف والتاريخ والمكان
- توفير تجربة مستخدم بسيطة وتفاعلية
- دعم الوضع الداكن (Dark Mode) لتحسين تجربة القراءة

---

## 🛠️ التقنيات المستخدمة

## 📄 صفحات الموقع

### 1. الصفحة الرئيسية (index.html)
- Navbar للتنقل بين الصفحات مع زر Dark Mode
- Slider لعرض الفعاليات البارزة (Carousel)
- قسم التصنيفات (Badges)
- قسم أحدث الفعاليات (Cards)
- Call to Action
- Footer شامل

### 2. صفحة الفعاليات (events.html)
- أدوات فلترة (بحث، تصنيف، تاريخ)
- عرض جميع الفعاليات بـ Cards
- تفاصيل لكل فعالية
- أزرار التنقل للتفاصيل

### 3. صفحة تفاصيل الفعالية (event.html)
- معلومات كاملة للفعالية
- خريطة ثابتة
- أزرار (إضافة للتقويم، مشاركة، حجز)
- قسم فعاليات مشابهة
- Modal للحجز

### 4. صفحة عن الدليل (about.html)
- نبذة عن المنصة
- الرؤية والرسالة
- فريق العمل
- سياسات النشر

### 5. صفحة اتصل بنا (contact.html)
- نموذج تواصل مع التحقق
- رسائل نجاح/خطأ
- معلومات التواصل

## ⚙️ التفاعلية (JavaScript)

تم تنفيذ الميزات التالية:

1. **فلترة الفعاليات**:
   - البحث النصي
   - التصنيف (رياضة/تقنية/ثقافة)
   - التاريخ
   - حفظ التفضيلات في localStorage

2. **Dark Mode**:
   - تبديل الوضع الداكن
   - حفظ التغيير في localStorage

3. **Scroll to Top**:
   - زر العودة للأعلى
   - إظهار عند التمرير

4. **نموذج الاتصال**:
   - التحقق من الحقول
   - عرض رسائل نجاح/خطأ
